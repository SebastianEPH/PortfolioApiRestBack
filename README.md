# Install

* Create `.env` in main folder

````env
PORT=5000
VERSION_API=v2.0
DB_SQL_USER=root
DB_SQL_PASSWORD=87654321
DB_SQL_HOST=localhost
DB_SQL_DATABASE=portfolio_db
DB_SQL_PORT=3306
````

* Import Database MySQL form folder `Database` => `portfolio-without-data.sql`
